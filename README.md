# flappybird

This repository contains a small, single-file Flappy Bird clone you can run locally in your browser.

Running locally

- Open [index.html](index.html) directly in your browser.
- Or serve the folder from a local HTTP server (recommended):

```bash
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```

With npm (uses npx http-server):

```bash
npm install
npm start
# then open http://localhost:8080
```

Deploying

- To publish on GitHub Pages, go to the repository Settings → Pages and publish from the `main` branch (root).

Enjoy! This is a tiny demo meant for local and static hosting.
